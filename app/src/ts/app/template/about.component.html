History

In Case Anybody is Interested

This is the third iteration of the Improv Database created by Shauvon McGill.

Shauvon buit the first version of the database over a weekend in 2008, probably instead of an important school project of some sort. It was originally an experiment in dumb UI and Asynchronous data manipulation with Javascript, which was still pretty new at the time. It looked, and worked, really dumb.

Look at how dumb this was. It actually still exists if you want to give it a shot. The database currently doesn't exist, but I think the easter eggs and the crazy user interface should still work. Good luck!

Around 2010 Shauvon purchased the ImprovDatabase.com domain to host the project, but it mostly sat untouched for another five years.

In early 2015, Shauvon began development of an entirely new app, built using modern technologies (and a usable interface). Around March of 2015, it was published online to the most minimal of fanfare.

A bit more usable!

Shauvon was not content, though. The interface was still a bit wonky and slow, and several features were just not as great as they could be. So, in late 2016, he sat down and completely rebuilt the thing again.

Here is Mark 3 as it first appeared in October, 2016. Not pictured: just how quickly and smoothly it loads.

The ImprovPlus app (nee The Improv Comedy Database) Mark IV is built with a pure Angular 2 front-end, connecting to a REST API written on NodeJS that interfaces with a PostgreSQL database. The application is hosted on Heroku. DNS and SSL services are provided by DNSimple. The SSL certificate currently ensuring your secure perusal of this page was provided by Comodo.
